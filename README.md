XCore
XCore — это мощный инструмент для настройки и управления прокси-серверами на базе Xray, NGINX и HAProxy. Проект предоставляет автоматизированную установку, управление пользователями, мониторинг и интеграцию с Cloudflare для выдачи SSL-сертификатов. XCore включает Bash-скрипт и скомпилированный бинарный файл для управления сервером и API-запросами.
Основные возможности

Автоматическая установка Xray, NGINX, HAProxy и зависимостей.
Управление пользователями (добавление, удаление, лимиты IP, продление подписки).
Интеграция с Cloudflare для автоматической выдачи и обновления SSL-сертификатов.
Мониторинг через Node Exporter и Prometheus.
Поддержка WARP для повышения безопасности соединений.
Резервное копирование и ротация конфигураций.
Статистика трафика и DNS-запросов.
Интерактивное меню на английском и русском языках.
API для управления статистикой и клиентами через бинарный файл.

Требования

Операционная система:
Ubuntu 20.04 или новее
Debian 10 или новее
CentOS 7 или новее


Права суперпользователя (root).
Доступ в интернет для загрузки зависимостей.
Домен, зарегистрированный в Cloudflare, с API-токеном или ключом.
Свободное место на диске: минимум 1 ГБ для бэкапов и логов.

Установка

Пример запуска установки с флагами (флаги работают только на установку или переустановку, больше нигде не используются):
sudo xcore --utils true --addu true --nginx true --xray true


Выберите язык (1 — English, 2 — Русский).
Выберите опцию 1 (Standard installation) для полной установки.
Введите данные: имя пользователя, пароль, домен, email и API-токен Cloudflare.
Подтвердите настройку SSH (опционально).



Использование
После установки XCore предоставляет интерактивное меню для управления сервером. Запустите:
xcore

Основные опции меню:

1. Стандартная установка: Устанавливает все компоненты (NGINX, HAProxy, Xray, мониторинг).
2. Восстановление из резервной копии: Восстанавливает конфигурации из архива.
4. Изменение домена: Обновляет доменное имя и сертификаты.
5. Перевыпуск сертификатов: Принудительно обновляет SSL-сертификаты.
6. Копирование сайта: Зеркалирует указанный сайт на сервер.
7. Размер директории: Показывает размер указанной папки.
8. Статистика трафика: Отображает статистику по годам, месяцам, дням или часам.
9. Смена языка: Переключает язык интерфейса.
X. Управление Xray: Управление клиентами, статистикой и DNS-запросами.


Обновление XCore
xcore --update

Конфигурационные файлы:

NGINX: /etc/nginx/
HAProxy: /etc/haproxy/
Xray: /usr/local/etc/xray/
Скрипты X Core: /opt/xcore/
Директория X Core go: /usr/local/xcore/xcore/
